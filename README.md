# Make_Gazebo_plugin
 
The repo aims to give have a simple hello gazebo plugin.
To test the repo:
**Don't forget to source the workspace{source ~/{worspace_name}/install/setup.bash}** 



In terminal-1:
```bash
   ros2 launch turtlebot3_gazebo turtlebot3_world.launch.py 

```
